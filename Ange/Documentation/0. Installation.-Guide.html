0. Installation.


Easiest way (autogenerating project)

In the simplest form, to start working with the ANGE library, first download and unpack it in to the desired project folder. Then run the console, go to that folder and execute in the console: 

premake5 <Your IDE>
Example command invoke: 
premake5 vs2017
This will generate a project for your IDE. Now all you have to do is run the IDE and start working on the generated project!


Typical way

Create a project in your IDE and then add the folder containing the ANGE source files to the compiler search folder list. Then make your linker link the static ange.lib library. This method also requires the installation of libraries: GLFW, FreeType2, GLIMG, GLLOAD, GLM and their linking.
